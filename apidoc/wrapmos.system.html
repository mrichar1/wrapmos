wrapmos.system package¶

Module contents¶

“Functions to handle the system (boot, sleep etc).

wrapmos.system.pin_callback(callback, pin=14)[source]¶

If pin is pulled down, call a callback.

This is useful to e.g. run particular code on boot.

Parameters:
  • callback – Method to be called.
  • pin – Pin to monitor.
wrapmos.system.stop_after(delay=60, action='reset', wake_after=600)[source]¶

Start a timer thread, which will restart/sleep the device after ‘delay’ secs.

Parameters:
  • delay – Delay before restart/sleep.
  • action – Reset or sleep the device.
  • wake_after – If sleeping, how long until wake (in seconds).
